## 1. 版本概述

v1.3 版本聚焦于"仪式感增强"和"用户参与度提升"，通过物理引擎、环境感知、AI角色扮演、数据可视化等功能，大幅提升应用的趣味性和沉浸感。

### 1.1 核心目标

- **仪式感升级**：通过物理引擎模拟真实摇铜钱体验
- **情境融合**：天气、音乐等环境因素融入占卜过程
- **智能互动**：AI 角色扮演，多轮对话深入解读
- **数据洞察**：历史运势可视化，满足用户自我探索需求
- **趣味学习**：游戏化学习六爻知识

### 1.2 功能优先级

| 优先级 | 功能模块 | 预计工时 |
|--------|----------|----------|
| P0 | 沉浸式"摇铜钱"物理引擎 | 高 |
| P1 | 本地"赛博占星师"角色扮演 | 中 |
| P1 | 个人运势可视化仪表盘 | 中 |
| P2 | 动态环境感知起卦 | 中 |
| P3 | "六爻"解谜小游戏 | 高 |
| P3 | 便携式"电子护身符" | 低 |

## 2. 功能需求

### 2.1 沉浸式"摇铜钱"物理引擎 (Physics-based Coin Tossing)

#### 2.1.1 功能描述

引入轻量级 2D 物理引擎，让用户通过真实的交互动作模拟传统"摇铜钱"起卦过程。

#### 2.1.2 交互设计

| 交互方式 | 描述 |
|----------|------|
| 鼠标摇动 | 快速点击或滑动模拟"摇动"动作 |
| 键盘摇动 | 按住空格键模拟持续"摇晃" |
| 物理模拟 | 三枚铜钱在虚拟碗中碰撞、翻滚、弹跳 |

#### 2.1.3 功能规格

| 项目 | 规格 |
|------|------|
| 物理引擎 | Matter.js（轻量级 2D 物理引擎） |
| 动画帧率 | 60 FPS |
| 铜钱模型 | 正反面纹理，支持旋转、碰撞检测 |
| 结果判定 | 根据物理碰撞结果自然落下显示正反面 |
| 元数据记录 | 记录摇动力度、轨迹、时长（供未来分析） |

#### 2.1.4 用户体验流程

```
点击"铜钱起卦"
    │
    ├── 显示虚拟铜钱和容器
    │
    ├── 用户开始摇动（鼠标滑动/按住空格）
    │   ├── 铜钱在容器中物理碰撞
    │   ├── 音效反馈（碰撞声）
    │   └── 视觉反馈（运动模糊、残影）
    │
    ├── 用户停止摇动
    │   └── 铜钱逐渐静止，显示结果
    │
    └── 记录结果（第N次，正反面组合）
```

#### 2.1.5 技术要点

- 完全本地计算，无需网络
- 摇动轨迹作为元数据存储，未来可分析"心流"状态
- 支持跳过动画直接出结果（快速模式）

### 2.2 动态环境感知起卦 (Context-Aware Divination)

#### 2.2.1 功能描述

利用本地系统信息和外部 API，让起卦过程融入当下情境，增加"天人感应"的玄学趣味。

#### 2.2.2 天气同步

| 项目 | 规格 |
|------|------|
| 数据来源 | 开源天气 API（如 Open-Meteo，免费无需 Key） |
| 获取方式 | 根据用户 IP 定位或手动输入城市 |
| 天气类型 | 晴、多云、雨、雷、雪、雾 |
| 功能联动 | 雷雨天自动推荐"震卦"相关提示，背景增加雨声白噪音 |

#### 2.2.3 音乐氛围

| 项目 | 规格 |
|------|------|
| 音频资源 | 内置古琴、编钟、笛子等中国风音效 |
| 卦象联动 | 根据得出的卦象自动播放对应意境背景音乐 |
| 离线支持 | 所有音频打包在应用内，确保离线可用 |

#### 2.2.4 环境感知元数据

记录每次起卦的环境信息：
- 天气状况
- 农历日期
- 节气
- 起卦时辰

### 2.3 本地"赛博占星师"角色扮演 (Local AI Roleplay Mode)

#### 2.3.1 功能描述

将 AI 解读从"工具型"升级为"角色型"，支持多轮对话深入解读。

#### 2.3.2 人格模式

| 模式 | 描述 | Prompt 风格 |
|------|------|-------------|
| 严谨易学大师 | 引用古籍，严肃分析 | 引用《周易》《增删卜易》原文，专业术语详尽 |
| 现代生活导师 | 用职场、情感角度通俗解读 | 结合现代生活场景，避免晦涩术语 |
| 神秘隐士 | 使用隐喻、诗歌风格回答 | 诗意化表达，留有解读空间 |

#### 2.3.3 多轮对话

| 功能 | 描述 |
|------|------|
| 上下文记忆 | AI 记住之前的对话内容 |
| 追问支持 | 用户可针对解读继续追问 |
| 会话管理 | 每次起卦创建独立会话，可回顾历史对话 |

#### 2.3.4 数据存储

- 所有对话历史加密存储在本地 SQLite
- 绝不上传云端，保护用户隐私

#### 2.3.5 交互流程

```
起卦完成后
    │
    ├── 选择 AI 人格模式（首次需设置，后续记住偏好）
    │
    ├── 输入问题/选择预设问题
    │
    ├── AI 生成初步解读
    │
    ├── 用户追问（支持多轮）
    │   ├── "大师，这个卦象问感情怎么看？"
    │   ├── "那事业方面呢？"
    │   └── "有什么需要注意的时间节点吗？"
    │
    └── 会话保存到历史记录
```

### 2.4 个人运势可视化仪表盘 (Personal Fortune Dashboard)

#### 2.4.1 功能描述

将历史记录转化为可视化图表，帮助用户发现运势规律。

#### 2.4.2 可视化模块

| 模块 | 描述 |
|------|------|
| 吉凶趋势图 | 统计过去一个月/一年的起卦结果，生成趋势曲线 |
| 词云分析 | 提取 AI 解读高频关键词（如"谨慎"、"合作"、"变动"），生成运势词云 |
| 卦象分布 | 统计出现频率最高的卦象 |
| 周期律报告 | 分析起卦频率与特定日期（农历初一、十五）的关系 |

#### 2.4.3 技术实现

| 项目 | 方案 |
|------|------|
| 图表库 | ECharts 或 Chart.js |
| 词云 | WordCloud2.js 或 Canvas 绘制 |
| NLP 分析 | 调用 Ollama 小模型提取关键词 |

#### 2.4.4 数据隐私

所有分析在本地完成，数据不出应用。

### 2.5 "六爻"解谜小游戏 (Hexagram Puzzle Mini-game)

#### 2.5.1 功能描述

将枯燥的六爻知识学习游戏化，增加趣味性。

#### 2.5.2 游戏模式

| 模式 | 描述 |
|------|------|
| 卦象连连看 | 展示卦象，限时拖拽阴阳爻还原成正确的 64 卦之一 |
| 断卦挑战 | 给出场景描述，让用户选择最合适的卦象，AI 评分解析 |
| 爻辞背诵 | 爻辞填空挑战 |

#### 2.5.3 成就系统

| 成就 | 解锁条件 |
|------|----------|
| 初窥门径 | 完成首次卦象连连看 |
| 登堂入室 | 断卦挑战正确率 80% 以上 |
| 易学大师 | 全部成就解锁 |

#### 2.5.4 数据存储

成就数据存储在本地，支持数据备份恢复。

### 2.6 便携式"电子护身符" (Portable Digital Talisman)

#### 2.6.1 功能描述

利用 Electron 特性，增加仪式感强的实用功能。

#### 2.6.2 桌面挂件

| 项目 | 描述 |
|------|------|
| 功能 | 极简透明窗口，显示当天"宜忌"或最近吉卦 |
| 交互 | 可拖动、可调整透明度、可关闭 |
| 自动启动 | 支持开机自启动 |

#### 2.6.3 一键生成壁纸

| 项目 | 描述 |
|------|------|
| 功能 | 根据用户抽到的吉卦，生成带卦象符号和签文的精美图片 |
| 技术 | Canvas 绘制，支持多种风格模板 |
| 分享 | 保存为本地图片，方便设为手机壁纸 |
